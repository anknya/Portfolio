ModelsGuild

Palo Alto Models Guild is an organization which provides models for educational facilities and artist groups in the Bay Area. But while it's been functioning for over 60 years, all the scheduling was still being done manually, which amounted to a lot of work, and left a lot of room for human error. So I designed a web application, which allows simple, efficient scheduling, while taking into account the users needs and preserving the values of the organization.

Treehouse

Treehouse Tribe is a San Francisco based organization that provides behavior therapy for children with learning differences. The program offers kids a compassionate learning experience by providing a safe space for children to interact with neurotypical kids & develop social skills while having fun! But since the scheduling of these playdates has been done over the phone, a 90 minute session could take up to 3 hours to schedule. We designed a mobile app that allows clients to schedule the playdates directly, as well as look for more friends themselves.

Smart Sales Shopping

We interviewed women in San Francisco and discovered that a lot of them are actually avoiding shopping at malls, because they simply don't have time for the them. So we came up with an app that would generate a shopping itinerary of sales which allows bargain hunting to be an efficient experience.
